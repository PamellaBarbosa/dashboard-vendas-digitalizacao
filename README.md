# Dashboard de Digitalização de Vendas

Projeto de análise de dados com foco em acompanhamento de vendas, desenvolvido a partir de base anonimizadas de um cenário real.

## Objetivo
Analisar o desempenho de vendas por filial e vendedor, identificando oportunidades de melhoria.

## Ferramentas
- Power BI
- SQL
- Google Sheets/ Excel
- Power Query

## Dados
Os dados utilizados são anonimizados e simulados, preservando a estrutura e as regras de negócio do cenário real.

## Resultado
Dashboard com indicadores de faturamento, percentual de digitalização e ranking por filial e vendedor.
